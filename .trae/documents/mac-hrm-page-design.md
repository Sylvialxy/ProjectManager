# Mac 端人力管理应用｜页面设计说明（Desktop-first）

## 全局设计（适用于所有页面）
- Layout：桌面优先，主内容区采用 CSS Grid（左侧导航 240px + 右侧内容自适应）；内容内部多用 Flex 布局。
- Breakpoints：
  - Desktop（>= 1280）：标准双栏布局。
  - Small Desktop（1024-1279）：左侧导航可折叠为图标栏。
- Meta：
  - 默认 Title：人力管理｜{PageName}
  - Description：录入项目投入与交付日期，甘特图查看占用与饱和度
- Global Styles（Tokens）：
  - 背景：#0B1220（深色）/ #F7F8FA（浅色，可选）
  - 主色：#3B82F6；强调：#22C55E；警示：#F59E0B；危险：#EF4444
  - 字体：系统字体（SF Pro/苹方优先），字号 12/14/16/20/24
  - Button：Primary/Secondary/Destructive；hover 提升亮度 6%；disabled 降低透明度 50%
  - 表格与卡片：圆角 10px；边框 #1F2937（深色）或 #E5E7EB（浅色）

---

## 页面 1：登录/加入
- Layout：居中单卡片（max-width 420px），背景为渐变或品牌色块。
- Meta：Title=登录/加入；OG 标题同 Title。
- Page Structure：顶部品牌区 + 表单区 + 辅助说明。
- Sections & Components：
  1. 品牌区：产品名、简短一句话价值描述。
  2. 登录表单：邮箱输入、发送登录链接/验证码按钮、错误提示。
  3. 邀请加入提示：若检测到邀请参数，展示“加入组织/团队”确认区。
  4. 会话状态：已登录则显示“继续进入总览”。

---

## 页面 2：总览（甘特图与饱和度）
- Layout：Grid 上方工具栏 + 下方左右分区（左：列表/筛选；右：甘特主视图）。
- Meta：Title=总览；Description 强调“甘特图与饱和度”。
- Page Structure：
  - Top Bar：时间范围、视图切换（个人/业务方向）、搜索、导出（如不做可隐藏占位）。
  - Body：左侧“对象列表”，右侧“时间轴甘特”。
- Sections & Components：
  1. 顶部工具栏：
     - 时间范围：周/月/自定义（日期选择器）。
     - 视图切换：Segmented Control（个人/业务方向）。
  2. 左侧列表区：
     - 在个人视图：成员列表（仅自己或可见成员），每行显示：姓名、当期饱和度%、超载标记。
     - 在业务方向视图：业务方向列表，每行显示：方向名、占用总小时、饱和度%。
  3. 右侧甘特图：
     - 时间轴：固定表头；按天/周刻度。
     - 条形块：项目阶段条；颜色按项目或阶段类型。
     - Tooltip：项目名、阶段名、开始/结束、交付日期、计划/实际小时、规则口径后的占用小时。
  4. 指标摘要卡：
     - 当期总占用、平均饱和度、超载人数/方向数。
- Interaction States：
  - 点击左侧某行高亮并滚动甘特到对应行。
  - 超载（>100%）用警示色；接近饱和（80-100%）用提示色。

## 页面 3：项目与阶段录入
- Layout：左侧项目列表 + 右侧详情编辑（Master-Detail）。
- Meta：Title=项目与阶段录入。
- Page Structure：
  - 左：项目列表（搜索、按业务方向筛选）。
  - 右：项目详情 Tabs（项目基本信息 / 阶段 / 投入）。
- Sections & Components：
  1. 项目列表：项目名、业务方向、交付日期、状态；新建按钮。
  2. 项目基本信息：项目名输入、业务方向选择、交付日期选择、保存/取消。
  3. 阶段管理：阶段表格（名称、类型、开始/结束、规则）；新增/编辑/删除；日期合法性提示（阶段区间与交付日期关系）。
  4. 投入录入：
     - 选择阶段后，显示成员投入表（成员、计划小时、实际小时、更新时间）。
     - 支持批量编辑（表格可编辑单元格）与“复制上一周期”（如启用）。

---

## 页面 4：规则与基础数据设置
- Layout：设置页常见两栏：左侧设置菜单，右侧表单/表格。
- Meta：Title=设置。
- Page Structure：菜单项包含：阶段投入规则 / 业务方向 / 成员与权限。
- Sections & Components：
  1. 阶段投入规则：规则表格（阶段类型、计算方式、系数、默认产能小时/天、启用）；新增/编辑弹窗。
  2. 业务方向：方向列表（名称、启用）；新增/停用。
  3. 成员与权限：成员表（邮箱、姓名、角色、状态）；邀请成员；授予/撤销管理员；停用成员。
